APP ICONS

Create two PNG images for the home screen icon:
- icon-192.png (192x192 pixels)
- icon-512.png (512x512 pixels)

Suggested design: Red "S" on dark background
- Background: #1a1a1a (dark gray)
- Letter: #c41e3a (Spartan red)

You can create these using any image editor or online tool.
The app will work without them, but won't have a custom icon on your home screen.
